React Hooks




useState

useState: Allows you to add local state to functional components.



useEffect

useEffect: Performs side effects in components, like fetching data or updating the DOM.



useId

useId: Generates a unique ID for accessibility and consistent server/client rendering.



useReducer

useReducer: Manages complex state logic using a reducer function and dispatch actions.



useRef

useRef: Creates a mutable reference that persists across renders, often used to access DOM elements.
